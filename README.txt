瓦酪露教會財務管理系統
========================
台灣基督長老教會瓦酪露教會 · 統一編號 87878809

【啟動方式】
雙擊 start.bat 或在命令提示字元執行：
  python app.py
然後在瀏覽器開啟 http://localhost:5000

【功能說明】
1. 儀表板    — 本月收支統計、走勢圖、快速操作
2. 現金簿    — 新增/編輯/刪除交易記錄，按月查看
3. 奉獻點算  — 奉獻點算表（面額統計、雙人簽署）
4. 月報表    — 收支月報（全年預算/本月/累計三欄）
5. 年度預算  — 設定各科目年度預算金額
6. 出納交接  — 出納交接確認表記錄
7. 查帳      — 查帳清單與紀錄（8項查核）
8. 設定
   - 會計科目（依長老教會總會標準預設56科）
   - 銀行帳戶管理
   - 各單位設定（團契/小組/事工）

【資料儲存】
資料存放於 instance/church.db（SQLite 資料庫）
建議定期備份此檔案。

【技術需求】
- Python 3.8+
- Flask, Flask-SQLAlchemy, openpyxl
